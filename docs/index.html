SCST Usermode Adaptation Project
A port of the SCST iSCSI storage server to run entirely in usermode on an unmodified Linux kernel
David A. Butterfield

iSCSI-SCST Storage Server Usermode Adaptation (PDF paper and diagrams)

The SCST Usermode Adaptation is based on scst-svn-7105-trunk and depends on these fixes to it: fixes patch

After applying the fixes_patch, this patch has the changes to allow running SCST in usermode: usermode_adaptation patch

The SCST Usermode Adaptation depends on this library: Multi-threaded Event Engine

After applying the above patches, this patch has experimental changes for Adaptive Nagle optimization and BLOCKIO/AIO: experimental patch
